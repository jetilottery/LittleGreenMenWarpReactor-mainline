Little Green Men Warp Reactor

Erkunden Sie interstellare Reichtümer in Little Green Men Warp Reactor!
Stimmen Sie 3 oder mehr identische benachbarte Planeten überein, um Cluster zu bilden. Die Cluster verschwinden und die Punktzahl für jeden Planeten steigt, während neue Planeten von oben herabfallen. Gewinnen Sie einen Preis, wenn Sie genügend identische Planeten sammeln. Lösen Sie den Warp-Bonus aus, wenn Sie 4 UFOs sammeln – Sie können sogar einen Sofortpreis gewinnen! Sammeln Sie 4 Asteroiden und lösen Sie den Spiral-Bonus aus: Die Planeten können ihre Position auf dem Raster ändern, wo sie zufällig eingesammelt werden können, um neue Cluster zu bilden und noch bessere Chancen auf einen Preisgewinn zu gewähren!

Spielanleitung

Ticketpreis

Zeigt den aktuellen Ticketpreis an.

Drücken Sie auf den linken Pfeil (–), um den Ticketpreis zu verringern.

Drücken Sie auf den rechten Pfeil (+), um den Ticketpreis zu erhöhen.

Drücken Sie auf KAUFEN, um den Ticketpreis vom Guthaben abzuziehen und das Spiel zu spielen.

Spielanleitung

Das Ziel von Little Green Men Warp Reactor ist es, 3 oder mehr identische benachbarte Planeten übereinzustimmen.

Wenn sich 3 oder mehr identische Planeten nebeneinander im Raster befinden, bilden sie Cluster. Die Cluster werden entfernt und die Punktzahl für jeden Planeten erhöht sich entsprechend. Sammeln Sie die erforderliche Anzahl von Planeten, um einen Preis zu gewinnen. Für jeden Planeten stehen mehrere Preise zur Verfügung.

Sammeln Sie 4 UFOs, um den Warp-Bonus auszulösen und zusätzliche Planeten zu sammeln. Sammeln Sie ein Stern-Symbol, um einen Sofortpreis zu gewinnen.

Sammeln Sie 4 Asteroiden, um den Spiral-Bonus auszulösen. Planeten können ihre Position auf dem Raster ändern, wo sie zufällig eingesammelt werden können oder neue Cluster bilden können.

Ein Bonusspiel wird nur ausgelöst, wenn keine Cluster mehr auf dem Raster sind.

Das Spiel endet, wenn keine Cluster mehr auf dem Raster sind und alle Bonusspiele abgeschlossen wurden. Eine Meldung zeigt das Ergebnis des Spiels an.

Regeln

Missbrauch oder Fehlfunktion annullieren alle Zahlungen und Spiele.

Gewinne werden in der ausgewählten Währung angezeigt.

Spielerentscheidungen, die während des Spiels getroffen werden, haben keinen Einfluss auf das Spielergebnis.
